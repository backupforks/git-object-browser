Step 13. remote add

$ git remote add gist https://gist.github.com/7074135.git

remote add コマンドでリモートリポジトリを登録しました。

Github で適当に作ったサンプルのリポジトリを gist という名前でリモートに追加しています。名前はなんでも構いません。

変更されたファイルは config だけです。 Prev と Next ボタンを使って、何が変わったのかを確認してください。

最後の [remote "gist"] が追加されています。url はリモートのURLです。fetch は、git fetch コマンドが、リモートのどのブランチからローカルのどのブランチに変更を取り込むかの設定です。コロン(:)の左がリモート、右がローカルです。

デフォルトではリモートの refs/heads/ にある全ブランチ(から参照されるオブジェクト)をローカルにコピーし、リファレンスを refs/remotes/gist/ 以下に反映します。

fetch 先頭のプラス記号は、リモートがローカルの Fast-forward の位置関係でなくても、上書きでローカルに反映することを意味しています。
