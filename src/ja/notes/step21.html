Step 21. stash pop index

$ git reset --hard HEAD

$ git stash pop --index

reset コマンドで作業ディレクトリを元に戻し、Step 18で待避した内容を --index オプションを付けて pop しました。

stash リファレンスとログファイルは削除されました。

--index オプションにより、index ファイルに Step 18 で add した内容が反映されています。index からたどって、a.txt に aaa が書き込まれているのを確認できます。--index を付けないとこの変更は起こりません。

ここには表示されていませんが、作業ディレクトリの bbb.txt には bbb と書かれています。
