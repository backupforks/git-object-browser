repo1 Step 2. clone

$ cd ..

$ git clone ./bare ./repo1

$ git clone ./bare ./repo2

$ cd repo1

bare リポジトリを clone して、repo1 と repo2 という2つのリポジトリを作成、repo1 に移動しました。

config ファイルにベアリポジトリのパスが書いてあります。origin というリモート名が自動的に付きました。

また、master ブランチに上流ブランチが設定されています。上流ブランチは、git pull や git push のデフォルトの動作に影響します。
