Step 24. rebase

$ git checkout -b step24

$ echo xxx > a.txt

$ git commit -am '...'

rebase コマンドを観察する準備として、step24 というブランチを作成し、a.txt に xxx と書き込みます。

$ git checkout master

$ echo aaa > b.txt

$ git commit -am '...'

$ echo bbb > b.txt

$ git commit -am '...'

master ブランチに戻り、b.txt に aaa と書き込みコミット、続けて b.txt に bbb と書き込みコミットしました。

step24 ブランチと master ブランチは枝分かれしています。step24 で1コミット、master で2コミットしました。
