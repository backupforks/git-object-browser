Step 23. tree 2

$ echo xxx > dir2/x.txt

$ git add dir2/x.txt

$ git commit -m '...'

dir2 に x.txt を追加しました。

dir2 の tree オブジェクトが差し替わっています。追加前はこうでした。

ディレクトリツリーのどこか一箇所が変わると、親の tree オブジェクトが変化し、親へ親へとさかのぼって、最終的にルートの tree オブジェクトが差し替わります。

ルート tree オブジェクトの ID (sha1) は、ディレクトリツリー全体を反映しています。1つでもファイルが違うと、別のIDになります。

このようにして、ルート tree オブジェクトの ID さえあれば、その時点での全ファイルを取り出すことが可能になっています。
