Step 15. merge remote

$ git merge remotes/gist/master -m '...'

step 14 で fetch した追跡ブランチを master にマージしました。

作成されたオブジェクトは2つです。gist.txt が追加された tree オブジェクトと、2つの parent を持つ commit オブジェクトが作成されました。ファイルは変わっていないので blob オブジェクトは作成されません。

git pull コマンドは、step 14 の fetch と、step 15 の merge を続けて実行します。追跡ブランチにリモートの変更を反映し、ローカルブランチにマージします。
