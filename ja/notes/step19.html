Step 19. stash 2

$ echo zzz > a.txt

$ git stash

今度は a.txt に zzz と書き込んで、git stash を実行しました。git add はしていません。

追加されたオブジェクトは全部で4つです。commit オブジェクトが2つ (WIP on master と index on master)、WIP on master からたどれる tree オブジェクト、zzz が書かれた blob オブジェクトです。

index on master が指している tree は以前からあるオブジェクトです。何も add していないので変化しません。

stash リファレンスは上書きされて、新しいWIP on master コミットを指すようになりました。

stash ログに、1行追加されました。stash リファレンスが上書きされてしまったので、Step 18 で追加された commit が記録されているのは、このログだけになっています。
